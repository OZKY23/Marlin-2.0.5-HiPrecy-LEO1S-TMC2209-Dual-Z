## HiPrecy LEO 1S

This repository fix some of the problems from the [original HiPrecy](https://github.com/HiPrecy)'s one and is intended to work with these specs:

* FYSETC F6 1.4 Motherboard
* TMC2209 Drivers
* Dual-Z Motors
* Bed Induction Probe
